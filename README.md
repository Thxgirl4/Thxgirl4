# 💫 About Me:
My journey as a developer is defined by building real-world solutions. Currently, I work in systems development within the agribusiness sector, where I have designed and implemented everything from APIs for geospatial data management to mobile applications used by end-users.

My core stack involves Python (Django/Flask) for a secure and high-performance backend, and JavaScript (React/React Native) for the frontend. Beyond just coding, I understand the value of data: I am proficient with SQL databases, data processing using Pandas, and system integration.

I am driven by continuous learning and the application of best practices such as Clean Code, version control with Git, and containerization.
